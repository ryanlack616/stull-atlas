## App name
Stull Atlas — Glaze Calculator

## Subtitle
Map-first glaze chemistry.

## Description (long)
Stull Atlas is a glaze calculation and Stull map exploration tool designed to feel like lab equipment—fast, clear, and repeatable.

• Calculate formulas and unity (UMF) with transparent diagnostics  
• Explore recipes on the Stull map with search and inspection  
• Compare recipes and track how changes move chemistry  
• Generate blend plans (line + triaxial) and export test sheets  
• Learn with Guided Adjustments and Explore-by-Example walkthroughs  
• Optional compatibility profiles for common calculators

Basic mode is friendly and accessible with big text and simple controls. Advanced mode unlocks beautiful data views, comparisons, blends, and checks—powered by one shared calculation engine.
