# Brand Voice & Style

## Voice keywords
Instrument-like • Calm • Precise • Helpful • Respectful • Not salesy

## Do
- Use plain language first, jargon second (“Show details”)
- Explain with cause/effect: “If you increase X, expect Y”
- Encourage testing and documentation

## Don’t
- Overpromise (“perfect”, “guaranteed”)
- Imply “AI magic” — prefer “guided, explainable steps”

## Microcopy patterns
- Errors: “Can’t compute yet” + one best next step
- Warnings: “Needs attention” + why + fix
- Actions: verbs first (“Normalize”, “Compare”, “Export test plan”)
