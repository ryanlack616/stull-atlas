# Social Media Images

Sized and formatted for each platform.

## Dimensions
| Platform     | Format          | Size        | File Pattern          |
|-------------|-----------------|-------------|----------------------|
| Twitter/X   | Post image      | 1200×675    | `twitter-*.png`      |
| Twitter/X   | Header          | 1500×500    | `twitter-header.png` |
| Instagram   | Square post     | 1080×1080   | `ig-square-*.png`    |
| Instagram   | Story           | 1080×1920   | `ig-story-*.png`     |
| LinkedIn    | Post image      | 1200×627    | `linkedin-*.png`     |
| Facebook    | Post image      | 1200×630    | `fb-*.png`           |
| Open Graph  | og:image        | 1200×630    | `og-*.png`           |

## Content Themes
1. **Product shot** — Screenshot of the Explorer with overlay text
2. **Data viz** — Isolated Stull chart cluster, visually striking
3. **Before/after** — "Guessing vs. knowing" side-by-side
4. **Testimonial** — Quote card from a user
5. **NCECA promo** — Conference-specific with booth number
